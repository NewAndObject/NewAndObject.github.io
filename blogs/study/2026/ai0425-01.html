# 总结

大模型提供商的价格体系非常复杂，且变动极为频繁（经常出现“价格战”或大幅度降价）。由于我无法实时联网获取最新的定价表（知识截止于2025年5月），以下为您整理的是截至2025年5月各主要提供商的主流模型定价模式、价格区间及重要趋势。

核心定价模式： 绝大多数大模型按Token计费（1个Token约等于0.75个中文词或0.25个英文单词），且通常输入（提问）和输出（回答）价格不同（输出更贵）。

# 一、 全球领先提供商（按上下文窗口和性能区分）

# 1. OpenAI (ChatGPT, API)

  • GPT-4o (最新多模态旗舰)：
    • 输入：约 $2.50 / 百万Token
    • 输出：约 $10.00 / 百万Token
  • GPT-4o-mini (高性价比小模型)：
    • 输入：约 $0.15 / 百万Token
    • 输出：约 $0.60 / 百万Token
  • o1系列 (推理/深度思考模型)：
    • o1-preview: 输入 $15，输出 $60 / 百万Token
    • o1-mini: 输入 $1.10，输出 $4.40 / 百万Token
  • 特点：价格最高，但生态最成熟。ChatGPT Plus订阅（$20/月）和Pro订阅（$200/月）是固定费用。

# 2. Anthropic (Claude)

  • Claude 3.5 Sonnet (主力模型，代码/长文强)：
    • 输入：$3.00 / 百万Token
    • 输出：$15.00 / 百万Token
  • Claude 3 Haiku (快速廉价版)：
    • 输入/输出：约 $0.25 / 百万Token（输入）/ $1.25 / 百万Token（输出）
  • 特点：输出价格较高，但在处理超长上下文（200K Token）时性能稳定。

# 3. Google (Gemini)

  • Gemini 1.5 Pro (当前主力)：
    • 输入（≤128K Token）：$1.25 / 百万Token
    • 输入（128K-1M Token）：$2.50 / 百万Token
    • 输出：$5.00 / 百万Token（1M上下文是最大优势）
  • Gemini 1.5 Flash (低成本、快速)：
    • 输入：$0.075 / 百万Token
    • 输出：$0.30 / 百万Token（极其便宜）
  • 特点：Flash版性价比极高，拥有100万Token上下文窗口。

# 4. Meta (Llama / 开源模型)

  • Llama 3.1 405B (开源最强，需通过云服务商调用)：
    • 通过Together AI或Groq等第三方：约 $2.00 - $5.00 / 百万Token（输入+输出混合）
  • Llama 3.1 70B / 8B：
    • 价格非常低，8B模型几乎免费，70B在Groq上约 $0.59 / 百万Token（输入）。
  • 特点：开源模型需要自己部署或通过第三方API，成本可控但需管理算力。

# 三、 关键价格趋势与对比

  1. 降价潮：自2024年中开始，国内大模型API价格下降了90%以上（如阿里、百度、DeepSeek）。现在的“价格战”已从百元/百万Token降至几元、几毛甚至免费。
  2. 国际 vs 国内：
    • 国际（OpenAI/Claude）：旗舰模型约 $2-15 / 百万Token（换算人民币约15-108元）。
    • 国内（DeepSeek/阿里/字节）：旗舰模型约 ￥0.5-20 / 百万Token（换算美元约$0.07-3）。
    • 差距：国内旗舰模型普遍比国际同等性能模型便宜 5-10倍。
  3. 输出 > 输入：几乎所有模型输出价格是输入的2-5倍（OpenAI GPT-4o约4倍；DeepSeek R1约8倍）。
  4. 缓存折扣：如果重复调用相同输入（如系统提示词），许多提供商（如DeepSeek、OpenAI）会给缓存命中部分 50%-90%的折扣。
  5. 批处理（Batch API）：OpenAI、Google等提供异步批处理，价格通常打 50%折扣，适合非实时任务。

# 四、 免费/低成本替代方案（2025年）

  • 完全免费：Google Gemini 1.5 Flash（API有免费额度，例如每分钟60次）、阿里通义千问Lite、百度ERNIE Speed。
  • 开源自部署：如果拥有GPU（如RTX 4090或Apple M系列芯片），可本地运行Llama 3.1 8B、Qwen2.5 7B/14B，零成本推理（仅用电费和硬件成本）。
  • 低成本第三方：Groq（专门为Llama系列提供极速推理，价格极低）、Together AI（开源模型集群）、Fireworks AI。
