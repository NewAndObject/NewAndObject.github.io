VuePress博客添加看板娘完整教程

悯 2026-05-03 14:30:00 VuePress看板娘Live2D

# 什么是看板娘

看板娘是一种基于 Live2D 技术的虚拟角色，通常会出现在网站的角落，为你的博客增添趣味和互动性。她可以对鼠标点击做出反应，切换不同的表情和动作。

# 前置条件

确保你的项目使用的是 VuePress 1.x 版本。本教程以 VuePress 1.5.0 + vuepress-theme-reco 1.5.5 为例。

# 安装插件

这里使用的是 vuepress-theme-reco 官方出品的看板娘插件：

npm install @vuepress-reco/vuepress-plugin-kan-ban-niang --save-dev
1

# 配置插件

在 .vuepress/config.js 中添加 plugins 配置：

module.exports = {
  // ...其他配置保持不变
  "plugins": [
    [
      "@vuepress-reco/vuepress-plugin-kan-ban-niang",
      {
        "theme": ["miku"],
        "clean": true,
        "modelStyle": {
          "position": "fixed",
          "left": "0px",
          "bottom": "0px",
          "opacity": "0.9",
          "zIndex": 99999
        }
      }
    ]
  ]
}

# 配置项说明

# 1. theme - 模型主题

可选的模型主题列表：

主题名 说明
miku 初音未来
whiteCat 白猫
blackCat 黑猫
koharu 小春
haru1 春 1
haru2 春 2
haruto 春人
izumi 泉
shizuku 雫
wanko 汪酱
z16 Z16

可以同时配置多个主题，第一个为默认显示，点击看板娘的切换按钮会随机切换：

"theme": ["miku", "koharu", "whiteCat"]
1

如果只配置一个主题，切换按钮会自动隐藏。

# 2. clean - 清洁模式

值 说明
true 隐藏所有按钮，只显示看板娘模型
false 显示所有功能按钮（切换、关闭、主页等）

# 3. modelStyle - 模型样式

通过 CSS 样式对象控制看板娘的位置和外观：

"modelStyle": {
  "position": "fixed",
  "left": "0px",      // 距离左侧的距离
  "bottom": "0px",    // 距离底部的距离
  "opacity": "0.9",   // 透明度
  "zIndex": 99999     // 层级
}

# 完整配置示例

module.exports = {
  "title": "我的博客",
  "description": "一个有看板娘的博客",
  "theme": "reco",
  "themeConfig": {
    // 主题配置...
  },
  "plugins": [
    [
      "@vuepress-reco/vuepress-plugin-kan-ban-niang",
      {
        "theme": ["miku", "koharu", "whiteCat"],
        "clean": false,
        "messages": {
          "welcome": "欢迎来到我的博客~",
          "home": "要回首页吗？",
          "theme": "想看看其他小伙伴吗？",
          "close": "不要走嘛~"
        },
        "modelStyle": {
          "position": "fixed",
          "left": "0px",
          "bottom": "0px",
          "opacity": "0.9",
          "zIndex": 99999
        },
        "messageStyle": {
          "left": "68px",
          "bottom": "190px"
        },
        "btnStyle": {
          "left": "0px",
          "bottom": "40px"
        },
        "width": 150,
        "height": 220
      }
    ]
  ]
}

# 常见问题

# 1. 看板娘不显示

  • 检查插件是否正确安装
  • 检查 config.js 中 plugins 配置格式是否正确
  • 打开浏览器开发者工具查看是否有资源加载失败的报错

# 2. 模型加载慢

看板娘的模型资源从 GitHub 加载，国内访问速度可能不稳定。可以：

  • 耐心等待首次加载
  • 使用 CDN 加速（需要下载插件源码并修改模型 URL）

# 3. 模型被其他元素遮挡

调整 zIndex 值，确保大于其他元素的层级：

"modelStyle": {
  "zIndex": 999999
}
1
2
3

# 4. 移动端不需要看板娘

看板娘在移动端会占用屏幕空间，建议通过 CSS 在移动端隐藏：

"modelStyle": {
  "position": "fixed",
  "left": "0px",
  "bottom": "0px",
  "opacity": "0.9",
  "zIndex": 99999,
  "@media (max-width: 768px)": {
    "display": "none"
  }
}

# 效果验证

运行开发服务器：

npm run dev
1

打开浏览器访问 http://localhost:8080，你应该能在页面左下角看到看板娘。

点击看板娘可以切换表情，点击底部按钮可以：

  • 切换主题：随机切换配置的模型
  • 关闭：隐藏看板娘
  • 主页：返回首页
  • 信息：查看插件信息
