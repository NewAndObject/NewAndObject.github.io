VuePress博客集成Giscus评论系统完整教程

悯 2026-05-03 14:00:00 VuePressGiscusGitHub

# 什么是Giscus

Giscus 是一个基于 GitHub Discussions 的评论系统。它的特点是：

  • 完全免费、开源
  • 无广告、无跟踪
  • 数据存储在 GitHub Discussions 中，完全由你自己掌控
  • 支持 Markdown 格式

相比传统的 Disqus 或 Valine，Giscus 更适合技术博客，因为评论者和评论数据都沉淀在 GitHub 生态里。

# 前置准备

# 1. 启用 GitHub Discussions

打开你的 GitHub 仓库，点击 Settings → Features → 勾选 Discussions

# 2. 安装 Giscus App

访问 https://github.com/apps/giscus，安装并授权访问你的仓库。

# 3. 获取配置参数

打开 https://giscus.app/zh-CN，按页面提示填写：

配置项 说明 示例
仓库 选择你的博客仓库 NewAndObject/NewAndObject.github.io
页面与讨论映射 推荐按 URL 路径匹配 pathname
Discussion 分类 选择或新建一个分类 Q&A
功能 按需勾选 reactions enabled

填写完成后，页面底部会生成配置代码，记下以下四个关键参数：

  • data-repo: 仓库名称
  • data-repo-id: 仓库 ID
  • data-category: 分类名称
  • data-category-id: 分类 ID

# 方案对比

方案 适用场景 优缺点

# 具体实施

# 1. 创建增强配置文件

在 .vuepress/enhanceApp.js 中写入以下代码：

export default ({ router, isServer }) => {
  if (!isServer) {
    router.afterEach(() => {
      setTimeout(() => {
        // 防止重复注入
        if (document.getElementById('giscus-container')) return
        
        // 查找文章内容区域
        const article = document.querySelector('.content__default') 
          || document.querySelector('.theme-reco-default-content')
          || document.querySelector('.page')
        
        if (article) {
          const container = document.createElement('div')
          container.id = 'giscus-container'
          container.style.marginTop = '4rem'
          container.style.padding = '0 2rem'
          article.appendChild(container)

          const script = document.createElement('script')
          script.src = 'https://giscus.app/client.js'
          script.setAttribute('data-repo', '你的用户名/你的仓库名')
          script.setAttribute('data-repo-id', '你的repoId')
          script.setAttribute('data-category', '你的分类名称')
          script.setAttribute('data-category-id', '你的categoryId')
          script.setAttribute('data-mapping', 'pathname')
          script.setAttribute('data-reactions-enabled', '1')
          script.setAttribute('data-emit-metadata', '0')
          script.setAttribute('data-input-position', 'bottom')
          script.setAttribute('data-theme', 'light')
          script.setAttribute('data-lang', 'zh-CN')
          script.setAttribute('data-loading', 'lazy')
          script.setAttribute('crossorigin', 'anonymous')
          script.async = true

          container.appendChild(script)
        }
      }, 1000)
    })
  }
}

# 关键点说明

代码片段 作用
router.afterEach() 路由切换后执行，确保页面跳转后评论也能加载
setTimeout(1000) 等待 DOM 渲染完成后再注入
!document.getElementById('giscus-container') 防止重复注入
data-loading: lazy 延迟加载，用户滚动到评论区域时才加载，提升性能

# 2. 创建可选的评论组件

在 .vuepress/components/Comments.vue 中创建组件：

<template>
  <div class="giscus-container">
    <div id="giscus-comments"></div>
  </div>
</template>

<script>
export default {
  name: 'Comments',
  mounted() {
    this.loadGiscus()
  },
  methods: {
    loadGiscus() {
      const script = document.createElement('script')
      script.src = 'https://giscus.app/client.js'
      script.setAttribute('data-repo', '你的用户名/你的仓库名')
      script.setAttribute('data-repo-id', '你的repoId')
      script.setAttribute('data-category', '你的分类名称')
      script.setAttribute('data-category-id', '你的categoryId')
      script.setAttribute('data-mapping', 'pathname')
      script.setAttribute('data-reactions-enabled', '1')
      script.setAttribute('data-emit-metadata', '0')
      script.setAttribute('data-input-position', 'bottom')
      script.setAttribute('data-theme', 'light')
      script.setAttribute('data-lang', 'zh-CN')
      script.setAttribute('data-loading', 'lazy')
      script.setAttribute('crossorigin', 'anonymous')
      script.async = true

      document.getElementById('giscus-comments').appendChild(script)
    }
  }
}
</script>

}

# 常见问题

# 1. 评论框不显示

  • 检查 GitHub Discussions 是否已启用
  • 检查 Giscus App 是否已安装并授权
  • 检查 data-repo-id 和 data-category-id 是否正确

# 2. 路由切换后评论重复加载

确保使用了 if (document.getElementById('giscus-container')) return 进行判断，防止重复注入。

# 3. 暗色模式适配

Giscus 支持暗色模式主题，修改 data-theme 为 dark 或使用 preferred_color_scheme 自动跟随系统：

script.setAttribute('data-theme', 'preferred_color_scheme')
1

# 4. 内容区域选择器不匹配

不同主题的内容区域 class 不同，可以通过浏览器开发者工具查看并修改选择器：

const article = document.querySelector('.你的主题内容区域class')
1

# 效果验证

运行开发服务器：

npm run dev
1

打开浏览器访问 http://localhost:8080，进入任意博客文章页面，滚动到最底部即可看到 Giscus 评论区。

首次访问时，需要 GitHub 登录才能发表评论。评论数据会实时同步到你的 GitHub Discussions 中。
