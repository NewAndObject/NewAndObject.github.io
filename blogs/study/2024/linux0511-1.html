# 安装步骤

KPlayer配置（官方文档地址：https://docs.kplayer.net/v0.5.8/config/play.html）
1.服务器安装linux系统（以ubuntu为例），远程服务器，参考命令：SSH ubuntu@121.5.138.218
2.远程安装KPlayer，指令：curl -fsSL get.kplayer.net | bash，在KPlayer文件夹里复制文件，执行：cp config.json.example config.json
3.安装播放器驱动软件，指令：sudo apt install mplayer
播放测试：mplayer /home/ubuntu/video/36728.mp4 正常会输出一大堆字符，其实测试不了什么
4.在创建存放视频的文件夹，例如：/home/ubuntu/video
创建指令：进入对于的文件夹 输入命令 mkdir video
5.上传视频
电脑打开cmd指令控制程序，输入scp E:\4.生活文件\1.手机\图片视频\OPPO_MP4\36728.mp4 ubuntu@121.5.138.218:/home/ubuntu/video
6.在kplayer文件夹里创建配置config.json文件，配置视频路径和推流地址
配置如下：
{
    "version": "2.0.0",
    "resource": {
        "lists": [
            "/home/ubuntu/video/36728.mp4"
        ]
    },
    "output": {
        "lists": [
            {
                "path": "rtmp://live-push.bilivideo.com/live-bvc/?streamname=live_19489836_13992780&key=ba06417c570cab41a44302e1509985ff&schedule=rtmp&pflag=1"
            }
        ]
    }
}
7.启动KPlyaer 
在kplyaer文件夹下运行： ./kplayer play start --daemon（这个命令是后台挂在那里，如果用：./kplayer play start ，就会退出命令界面就会停止直播）

8.关闭KPlyaer
在kplyaer文件夹下运行：./kplayer play stop

备注：
1.为什么要安装mplayer
-因为不安装就无法播放，因为ubuntu系统可能不带播放器解码器，没有视频解码器，系统就无法解码，那就播放不了了
2.为什么我的哔哩哔哩推流地址后，启动推流报错
-在config.json 里output的lists配置可能有问题，你要在开播设置里开启直播，开启后，会有服务器地址和串流密钥，需要你将服务器地址和串流密钥合并放入lists里配置就好了
3.怎么循环播放
-在config.json里配置play的player_model的loop配置，参考入下
{
  "play": {
    "play_model": "loop"
  }
}
4.为什么我都配置好了，控制面板报错
-可能是没有用户点击你对直播间，你可以自己进去看看试试
-可能是上传的视频不完全，删除重新上传一遍试试
5.为什么我再次连接ubuntu报错
-可能是ubuntu服务器有问题或者你本地的连接缓存有问题，你可以cmd试试清除ssh连接缓存：ssh-keygen -R 121.5.138.218，再重新连接试试：SSH ubuntu@121.5.138.218

6.常用linux指令
-创建文件夹 mkdir 文件夹名称
-进入文件夹 cd 文件夹名称
-返回上一层文件夹路径 cd ..
-编辑文件 vim 文件名称
-编辑文件夹进入编辑模式 shift+a
-编辑完成保存 按esc退出编辑模式，再输入 ：wq
-上传本地文件到服务器 scp 本地文件路径 ubuntu@121.5.138.218:服务器文件夹路径
-删除文件 定位到要删除的文件夹下 输入：rm 文件名称
