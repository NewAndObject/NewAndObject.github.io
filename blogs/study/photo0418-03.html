# 问题表

# 为什么改变构图后相机接受到的光线会发生变化

  • 影响源
    • 发光源发光强度
    • 受光体反射率（例如黑色吸光，反射率低）

# 测光表一定要放在中间吗？什么时候不放在中间

  • 首先了解照片亮度如何被定义的
    • 一个数码照片，把所有色彩排除，留下了的就是亮度，这个亮度用黑白灰的关系表达
    • 一张照片的亮度就是它的平均亮度，而曝光表就是来描述这个照片的平均亮度的，曝光表的中间值就是亮度标准中的50%灰（中性灰）

# 难道所有的照片都要拍成中性灰吗？请了解环境亮度

  • 影响亮度

# 为什么是 18%

- 反射率在18%的物体，表现为不亮不暗的中等亮度（中等亮度就是曝光表的中间值）
- 我们生活中所有物体的反射率平均为18%

  • 什么时候不放在中间

    • 在雪地，受光均匀明亮时，但是平均反射率高于18%，这个时候就需要把曝光表调高

    • 拍摄一串葡萄，受光均匀时，葡萄的反射率低于18%，这个时候就需要曝光表调低一点 低于的照片： 高于的照片：

    • 雾蒙蒙群山的蓝调情况下也需要曝光表调低

# 什么是白加黑减

  • 什么是做曝光补偿
    • 即白加黑减调节曝光表高低的过程
  • M挡有曝光补偿吗
    • 有，即你调节三要素的过程

# 测光模式有什么用？

  • 在光线非均匀情况下需要用到测光模式

  • 平均测光

    • 默认测量全图的平均亮度

  • 中央（重点）测光

    • 测中央区域的平均亮度

  • 点测光

    • 测量聚焦点的亮度

  • 举例

    • M挡模式时 拍同一物体，无论用那种测光模式，画面不会变化，仅仅显示哪个区域的亮度而已，因为画面只和曝光三要素有关系
    • 半自动，A挡或S挡时，这个时候，有一个是自动的，在A挡时，当你使用某个测光模式时，相机会自动调整快门，把你的照片调整为中性灰（即曝光表中间）

# 点测管可以解决逆光人像问题吗？

  • 不能，因为亮度差异太大，远远超过相机的容纳范围（宽容度）
  • 如何解决：缩小亮度差异
    • 给人补光，提高人的亮度
    • 后期给人物提亮亮度（这个不太好）
    • 其他情况，人物背光，但是不需要拍亮人物，这个时候就点测人脸就可以了（注意白加黑减）

# 天空很亮，地面很暗照片怎么拍

  • 风光场景，亮度太高了，就用滤镜把天空压暗
  • 拍摄多张不同亮度的照片，后期合成到一起
  • 矩阵测光（尼康）/评价测光（佳能）/多重测光（富士）
    • 原理：差不多是平均测光和点测光的结合版。不仅会关注点的亮度，还会照顾附近区域的亮度。
