# 介绍

# M档-手动挡

  • 可以自己调整光圈和快门，但是拍摄效率低

# A挡-光圈优先（半自动挡）

  • 自己调整光圈，快门自动不能调整
  • 目的：拍摄人像，需要虚化情况

# S挡-快门优先（半自动挡）

  • 自己调整快门，光圈自动不能调整
  • 目的：拍摄移动物体，例如：宠物、动物、体育运动和打鸟

# P挡-程序自动

  • 光圈自动，快门自动，但是，快门和光圈可以调整，例如婚礼、运动会等

# 图片讲解

# ISO自动

  • 当快门优先光圈自动时，你快门调整后，光圈自动调整到最大时还不够（感光度过低），这个时候，可以设置ISO自动，这样，就不会太暗
